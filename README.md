# 政务督查办管理系统 java333

## 项目概述

政务督查办管理系统是基于 SpringBoot 技术开发的一套适用于政府督查办公的应用系统。系统旨在提高政务工作的执行效率与监督力度，通过信息化手段实现政务工作的规范化、透明化。

## 技术栈

### 后端

- SpringBoot
- MyBatis

### 前端

- Vue

### 数据库

- MySQL 5.7

### 开发工具

- IntelliJ IDEA

## 功能模块

系统主要包括以下功能模块：

- 超级管理员模块
- 公民用户模块
- 监督人员模块
- 政务人员模块

各模块具体功能如下：

- 超级管理员：系统管理、用户管理、权限分配等。
- 公民用户：信息查询、事务申报等。
- 监督人员：督查任务管理、进度报告等。
- 政务人员：政务处理、工作汇报等。

## 系统角色

系统定义了以下角色：

- 超级管理员
- 公民用户
- 监督人员
- 政务人员

## 运行环境

- JDK 1.8 或以上版本
- MySQL 5.7
- 适用主流浏览器

## 部署步骤

1. 准备好运行环境，包括 JDK 和 MySQL。
2. 导入数据库文件，建立数据库。
3. 部署后端服务，运行 SpringBoot 应用。
4. 部署前端应用，确保前端能够访问到后端接口。
